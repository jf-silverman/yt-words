About This Website

I'm a data scientist with a background spanning environmental modeling, healthcare diagnostics, and clean energy analytics. One recurring theme across all of it: building automated pipelines to turn raw, messy data into something you can actually query and learn from.

This site is a personal project that applies those skills to a question I've always had as a Mad Money listener: does Jim Cramer's conviction level actually predict his accuracy? Does segment matter — is he sharper in his prepared Opening Commentary than in the Lightning Round? Which sectors does he call best?

To answer those questions rigorously, I built the infrastructure from scratch: a nightly pipeline that scrapes YouTube transcripts, extracts structured stock calls using Claude AI (with a carefully engineered prompt to minimize hallucination), tracks forward returns in a SQLite database, and publishes everything to this fully client-side GitHub Pages site. The Analytics tab is where the data starts to tell a story.

I'm a fan of Jim's — his show gave me the confidence to start investing, and I think he's one of the best financial educators working today. This site isn't a hit piece; it's a measuring stick, built for fun and to keep my data engineering skills sharp.

Disclaimer: This site is for educational purposes only — not investment advice. AI-generated analysis may contain errors. I'm not a financial advisor.
